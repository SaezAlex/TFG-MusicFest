# 🎸 MusicFest - App de Gestión de Festivales

Aplicación móvil nativa para Android diseñada para la gestión integral y experiencia de usuario en festivales de música. 

> **Nota del desarrollador:** Este proyecto nació originalmente como Trabajo de Fin de Grado (TFG) colaborativo para el ciclo superior de Desarrollo de Aplicaciones Multiplataforma (DAM). Actualmente, el repositorio se mantiene en **desarrollo activo**, escalando la arquitectura base y añadiendo nuevas funcionalidades de forma individual.

## 🚀 Características Principales

*   **Autenticación de Usuarios:** Sistema de registro y login seguro.
*   **Gestión de Cartelera:** Visualización de artistas, horarios y escenarios.
*   **Organizar el viaje:** Mediante botones podra el usuario buscarse alojamiento y ver la ruta hasta el festival.


## 🛠️ Stack Tecnológico

El proyecto está construido siguiendo buenas prácticas de desarrollo móvil:

*   **Lenguaje:** Kotlin
*   **Entorno:** Android Studio / Gradle
*   **Backend & Base de Datos:** Firebase (Authentication & Cloud Firestore/Realtime Database)

## 🚧 Próximas Mejoras (Roadmap)

Como parte de la evolución continua del proyecto, estoy trabajando en implementar:

*   Integración de pasarela de pago simulada para compra de entradas.
*   Mapa interactivo del recinto del festival.
*   Sistema de notificaciones push para avisos de inicio de conciertos.
